Fast-Food-Fast (3Fs) API - Version 1

Welcome to 3Fs API documentation home page

Test the below api endpoints in POSTMAN

API EndPoint User Functionality
GET /api/v1/menu admin add new food-item and price on menu available for order on a given day
GET /api/v1/client/menu client Get the menu of the meals available for order and their respective prices
POST /api/v1/orders Client Place new order
GET /api/v1/orders admin Get all the available orders
GET /api/v1/orders/orderID admin Get a specific order
GET /api/v1/client/orders/orderID client Client can get his/her specific order to check status
PUT /api/v1//orders/orderID admin Update status of specific order
DELETE /api/v1/orders/orderID admin Remove completed order
